Attendio 👑

Smart Attendance System using Face Recognition & Web Stack

Attendio is a cutting-edge attendance system designed to simplify and automate tracking using face recognition technology, with a smooth frontend-backend integration.
Perfect for classrooms, workplaces, or any organization needing fast, secure attendance tracking.

Features ✨
Face Recognition: Instantly identify students/employees using OpenCV & face_recognition.
Real-Time Attendance: Track attendance live with webcam input.
Database Storage: Store and manage attendance data in MySQL.
Web Interface: Easy-to-use frontend built with HTML, CSS, JS.
Secure & Efficient: Configurable via .gitignore & requirements.txt for clean deployment.
